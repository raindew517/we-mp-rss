# 测试用例 - 模板示例

## 用例1: 基本变量替换
- 输入数据:
  - name: "张三"
  - level: "普通会员"
- 预期输出:
  欢迎您，张三！
  您的会员等级是：普通会员

## 用例2: VIP会员条件判断
- 输入数据:
  - name: "李四"
  - level: "VIP"
- 预期输出:
  欢迎您，李四！
  您的会员等级是：VIP
  您是我们的尊贵VIP会员，享受专属优惠！

## 用例3: 订单信息嵌套变量
- 输入数据:
  - name: "王五"
  - level: "高级会员"
  - order:
    - id: "ORD12345"
    - product: "笔记本电脑"
    - quantity: 2
    - price: 5000
- 预期输出:
  欢迎您，王五！
  您的会员等级是：高级会员
  您的订单信息：
  订单号：ORD12345
  商品：笔记本电脑
  数量：2
  总价：10000

## 用例4: JSON嵌套数据
- 输入数据:
  - feed:
    - mp_name: "科技新闻"
  - articles:
    - title: "人工智能新突破"
      publish_time: "2025-10-24"
    - title: "量子计算进展"
      publish_time: "2025-10-23"
- 预期输出:
{
 "feed": "科技新闻",
"articles":[
    {"title": "人工智能新突破", "pub_date": "2025-10-24"},
    {"title": "量子计算进展", "pub_date": "2025-10-23"}
]
}

  Your membership level is: Gold Member

## 用例8: 复杂嵌套结构
- 输入数据:
  - user:
    - name: "周八"
    - address:
      - city: "北京"
      - district: "朝阳区"
    - orders:
      - id: "ORD67890"
        items:
          - name: "手机"
            price: 3999
          - name: "耳机"
            price: 599
- 预期输出:
  用户信息：
  姓名：周八
  地址：北京朝阳区
  订单：ORD67890
  商品清单：
  - 手机 (3999元)
  - 耳机 (599元)
  总金额：4598元
